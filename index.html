Ubuntu
"Linux for Human Beings"
Ubuntu là bản phân phối Linux phổ biến nhất thế giới, được phát triển bởi Canonical. Thiết kế hướng đến người dùng phổ thông với giao diện thân thiện, cài đặt dễ dàng và hỗ trợ phần cứng rộng rãi.

Arch Linux
"Keep It Simple, Stupid"
Arch Linux là bản phân phối tối giản, rolling release, được cài đặt thủ công từ đầu. Triết lý KISS cho phép người dùng xây dựng hệ thống hoàn toàn theo ý muốn.

03 — Cài đặt

Hướng dẫn
cài đặt Linux

Ubuntu — Cài đặt dễ dàng với GUI installer Trải nghiệm thân thiện ngay từ bước đầu tiên
📷 Ảnh từ: Unsplash.com
Ubuntu Ubuntu có GUI installer thân thiện nhất trong giới Linux. Thời gian cài đặt khoảng 15–30 phút, phù hợp cho mọi trình độ.

1
Tải Ubuntu ISO & tạo USB bootable
Chuẩn bị
# Tải ISO mới nhất từ trang chủ Ubuntu:
# https://ubuntu.com/download/desktop
# Phiên bản: Ubuntu 25.04 (Plucky Puffin)

# Tạo USB bootable trên Linux:
dd bs=4M if=ubuntu-25.04-desktop-amd64.iso \
  of=/dev/sdX conv=fsync status=progress

# Trên Windows: dùng Rufus (https://rufus.ie)
# Chọn: GPT partition, FAT32, chế độ ISO

# Trên macOS:
diskutil unmountDisk /dev/diskX
sudo dd if=ubuntu-25.04.iso of=/dev/rdiskX bs=4m
2
Boot vào Ubuntu Live & Cài đặt
GUI Installer
# Khởi động máy, nhấn F2/F12/Delete để vào BIOS
# Chọn boot từ USB — Ubuntu Live Desktop sẽ khởi động

# Nhấp đúp vào icon "Install Ubuntu" trên desktop
# Hoặc chạy installer từ Applications

# Các bước trong GUI installer:
# ┌─────────────────────────────────────────────┐
# │  1. Chọn ngôn ngữ (Vietnamese / English)    │
# │  2. Keyboard layout                          │
# │  3. Loại cài đặt:                            │
# │     ○ Normal install (Khuyến nghị)           │
# │     ○ Minimal install (Nhẹ hơn)             │
# │  4. Cài third-party drivers? → Tích vào ✓   │
# │  5. Disk setup:                              │
# │     ○ Erase disk and install (Đơn giản)      │
# │     ○ Manual partitioning (Nâng cao)         │
# │  6. Timezone → Asia/Ho_Chi_Minh              │
# │  7. Tạo tài khoản người dùng                 │
# └─────────────────────────────────────────────┘
3
Phân vùng thủ công (tuỳ chọn)
Advanced
# Nếu cài song song với Windows hoặc muốn tự phân vùng:
# ┌────────────┬──────────┬──────────────┬────────┐
# │ Partition  │   Size   │  Mount Point │  Type  │
# ├────────────┼──────────┼──────────────┼────────┤
# │ /dev/sda1  │  512 MB  │ /boot/efi    │ FAT32  │
# │ /dev/sda2  │  1 GB    │ /boot        │ ext4   │
# │ /dev/sda3  │  RAM×2   │ swap         │ swap   │
# │ /dev/sda4  │  30+ GB  │ /            │ ext4   │
# │ /dev/sda5  │  Còn lại │ /home        │ ext4   │
# └────────────┴──────────┴──────────────┴────────┘

# Hệ thống file được khuyến nghị:
# ext4 — phổ biến, ổn định
# btrfs — hỗ trợ snapshots, mới hơn
4
Sau khi cài — Cấu hình hệ thống
Post-install
# Cập nhật hệ thống ngay sau khi cài xong:
sudo apt update && sudo apt upgrade -y

# Cài các drivers cần thiết:
sudo ubuntu-drivers autoinstall

# Cài các phần mềm thiết yếu:
sudo apt install -y \
  curl wget git vim build-essential \
  htop fastfech vlc \
  gnome-tweaks chrome-gnome-shell

# Bật Flatpak (kho ứng dụng rộng hơn):
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub \
  https://flathub.org/repo/flathub.flatpakrepo

# Kiểm tra hệ thống:
fasfetch
lsb_release -a
5
Cài Development Environment
Dev Setup
# Cài Node.js (qua nvm):
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
nvm install --lts

# Cài Python tools:
sudo apt install python3 python3-pip python3-venv -y
pip3 install --user pipx

# Cài Docker:
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# Cài VS Code:
snap install code --classic

# Hoặc từ .deb package:
wget https://code.visualstudio.com/sha/download?build=stable&os=linux-deb-x64
sudo dpkg -i code_*.deb
Tip Ubuntu có GUI Software Center — bạn có thể cài ứng dụng bằng cách click chuột mà không cần terminal. Truy cập "Ubuntu Software" từ Applications.
Arch Linux — Xây dựng hệ thống từ nền tảng Terminal, archinstall và sự kiểm soát hoàn toàn
📷 Ảnh từ: Unsplash.com
Archinstall Archinstall là script cài đặt chính thức tích hợp trong ISO Arch Linux (từ 2021). Cung cấp TUI giúp cài đặt nhanh hơn nhiều.

1
Tải ISO & tạo USB bootable
Chuẩn bị
# https://archlinux.org/download/
dd bs=4M if=archlinux-*.iso of=/dev/sdX conv=fsync oflag=direct status=progress
# Windows: dùng Rufus, chọn DD Write mode
2
Boot & kết nối mạng
BIOS/UEFI
ping -c 3 archlinux.org
# WiFi:
iwctl
  station wlan0 scan
  station wlan0 connect "Tên WiFi"
3
Chạy Archinstall
TUI
archinstall

# Cấu hình trong menu:
# Mirrors        → Vietnam/Asia
# Locales        → vi_VN.UTF-8
# Disk config    → Best-effort erase
# Bootloader     → GRUB
# Profile        → Desktop (KDE/GNOME)
# Audio          → Pipewire
# Network        → NetworkManager
# Timezone       → Asia/Ho_Chi_Minh
4
Sau cài đặt
Post-install
sudo pacman -Syu
# Cài AUR helper:
git clone https://aur.archlinux.org/yay.git
cd yay && makepkg -si
⚠ Cảnh báoThực hiện mọi thao tác qua terminal. Backup dữ liệu trước khi bắt đầu!

1
Kiểm tra boot mode & mạng
Chuẩn bị
ls /sys/firmware/efi/efivars  # UEFI check
ping -c 3 archlinux.org
timedatectl set-ntp true
2
Phân vùng ổ đĩa
Disk
lsblk
cfdisk /dev/sda
# EFI: 512MB | Swap: 8-16GB | Root: còn lại
mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
mkfs.ext4 /dev/sda3
3
Mount & cài base system
Pacstrap
mount /dev/sda3 /mnt
mkdir -p /mnt/boot/efi && mount /dev/sda1 /mnt/boot/efi
swapon /dev/sda2
reflector --country Vietnam,Singapore --sort rate --save /etc/pacman.d/mirrorlist
pacstrap -K /mnt base linux linux-firmware linux-headers \
  base-devel networkmanager vim sudo grub efibootmgr
4
Cấu hình hệ thống
Config
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt
ln -sf /usr/share/zoneinfo/Asia/Ho_Chi_Minh /etc/localtime
hwclock --systohc
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf
echo "archlinux" > /etc/hostname
5
User & Bootloader
GRUB
passwd
useradd -m -G wheel -s /bin/bash myuser
passwd myuser
visudo  # bỏ comment %wheel ALL=(ALL:ALL) ALL
grub-install --target=x86_64-efi --efi-directory=/boot/efi
grub-mkconfig -o /boot/grub/grub.cfg
systemctl enable NetworkManager
exit && umount -R /mnt && reboot
6
Cài Desktop Environment
GUI
# KDE Plasma (khuyến nghị):
sudo pacman -S plasma kde-applications sddm
sudo systemctl enable sddm

# GNOME:
sudo pacman -S gnome gnome-extra gdm
sudo systemctl enable gdm

# XFCE (nhẹ):
sudo pacman -S xfce4 xfce4-goodies lightdm
sudo systemctl enable lightdm
