🟠
Ubuntu
// "Linux for Human Beings"
Ubuntu là bản phân phối Linux phổ biến nhất thế giới, được phát triển bởi Canonical. Thiết kế hướng đến người dùng phổ thông với giao diện thân thiện, cài đặt dễ dàng và hỗ trợ phần cứng rộng rãi. Lý tưởng cho người mới bắt đầu với Linux hay môi trường doanh nghiệp.
PHÁT HÀNH 2004 — Canonical
CƠ SỞ Debian
PACKAGE MGR APT / Snap
RELEASE LTS + Rolling
ĐỘ KHÓ Dễ — Beginner
DESKTOP MẶC ĐỊNH GNOME
PHIÊN BẢN MỚI NHẤT Ubuntu 25.04
🔵
Arch Linux
// "Keep It Simple, Stupid"
Arch Linux là bản phân phối tối giản, rolling release, được cài đặt thủ công từ đầu. Triết lý KISS (Keep It Simple) cho phép người dùng xây dựng hệ thống hoàn toàn theo ý muốn — chỉ có những gì bạn thực sự cần. Dành cho người dùng muốn học sâu về Linux.
PHÁT HÀNH 2002 — Judd Vinet
CƠ SỞ Độc lập
PACKAGE MGR Pacman / AUR
RELEASE Rolling Release
ĐỘ KHÓ Khó — Advanced
DESKTOP MẶC ĐỊNH Không có (tuỳ chọn)
PHIÊN BẢN Rolling (mới nhất liên tục)

// 03 — Cài đặt Arch Linux

Hướng dẫn
cài Arch Linux

// Archinstall là gì? Archinstall là script cài đặt chính thức được tích hợp trong ISO Arch Linux (từ 2021). Nó cung cấp giao diện TUI (text-based UI) giúp cài đặt nhanh chóng hơn nhiều mà không cần nhớ từng lệnh.

1
Tải và tạo USB bootable
Chuẩn bị
# Tải ISO từ trang chủ: https://archlinux.org/download/
# Tạo USB bootable trên Linux:
dd bs=4M if=archlinux-*.iso of=/dev/sdX conv=fsync oflag=direct status=progress

# Trên Windows: dùng Rufus hoặc Balena Etcher
# Chọn chế độ DD Write khi dùng Rufus
2
Boot vào Arch ISO
BIOS/UEFI
# Khởi động máy, vào BIOS/UEFI (F2, F12, Delete...)
# Tắt Secure Boot nếu cần
# Chọn boot từ USB

# Sau khi vào live environment, kiểm tra network:
ping -c 3 archlinux.org

# Nếu dùng WiFi:
iwctl
  # Trong iwctl shell:
  station wlan0 scan
  station wlan0 connect "Tên WiFi"
3
Chạy Archinstall
TUI Installer
# Đơn giản chỉ cần gõ lệnh này:
archinstall

# Các tuỳ chọn cần cấu hình trong menu:
# ┌─────────────────────────────────────────┐
# │  Archinstall guided installer            │
# ├─────────────────────────────────────────┤
# │  Mirrors           → Chọn Vietnam/Asia   │
# │  Locales           → vi_VN.UTF-8         │
# │  Disk config       → Best-effort erase   │
# │  Bootloader        → GRUB (recommended)  │
# │  Swap              → zram (recommended)  │
# │  Hostname          → archlinux           │
# │  Root password     → Đặt mật khẩu        │
# │  User account      → Tạo user mới        │
# │  Profile           → Desktop environment  │
# │  Audio             → Pipewire            │
# │  Network           → NetworkManager      │
# │  Timezone          → Asia/Ho_Chi_Minh    │
# └─────────────────────────────────────────┘
4
Chọn Desktop Environment
Profile
# Trong mục "Profile" → "Desktop":
# Chọn Desktop Environment tuỳ thích:

GNOME     → Hiện đại, mượt, giống macOS (nặng hơn)
KDE Plasma → Đẹp, nhiều tùy chỉnh, nhẹ hơn GNOME
XFCE      → Nhẹ, nhanh, phù hợp máy cấu hình thấp
Hyprland  → Tiling WM, cực đẹp với animations
i3        → Tiling WM, nhẹ, chỉ keyboard

# Khuyến nghị người mới: KDE Plasma hoặc GNOME
5
Cài đặt & Khởi động lại
Finish
# Sau khi cấu hình xong, chọn "Install"
# Quá trình cài đặt mất khoảng 5-15 phút tuỳ tốc độ mạng

# Sau khi cài xong, chọn "Reboot"
# Rút USB ra, boot vào hệ thống mới!

# Sau khi vào hệ thống, cập nhật toàn bộ packages:
sudo pacman -Syu

# Cài AUR helper (yay):
git clone https://aur.archlinux.org/yay.git
cd yay && makepkg -si
⚠ Cảnh báo Cách này yêu cầu bạn thực hiện mọi thao tác thủ công qua terminal. Đọc kỹ từng bước trước khi thực hiện. Backup dữ liệu quan trọng trước khi bắt đầu!

1
Kiểm tra kết nối mạng & Boot mode
Chuẩn bị
# Kiểm tra UEFI hay BIOS:
ls /sys/firmware/efi/efivars
# Nếu thư mục tồn tại → UEFI mode

# Kiểm tra internet:
ping -c 3 archlinux.org

# Kết nối WiFi nếu cần:
iwctl
  device list
  station wlan0 scan
  station wlan0 get-networks
  station wlan0 connect "SSID"

# Đồng bộ thời gian:
timedatectl set-ntp true
2
Phân vùng ổ đĩa
Disk Partitioning
# Liệt kê các ổ đĩa:
lsblk
# hoặc fdisk -l

# Phân vùng với cfdisk (dễ hơn):
cfdisk /dev/sda   # hoặc /dev/nvme0n1

# Sơ đồ phân vùng cho UEFI (khuyến nghị):
# ┌────────────┬──────────┬────────────┐
# │ Partition  │   Size   │    Type    │
# ├────────────┼──────────┼────────────┤
# │ /dev/sda1  │  512 MB  │ EFI System │
# │ /dev/sda2  │  8-16 GB │ Linux swap │
# │ /dev/sda3  │  Còn lại │ Linux fs   │
# └────────────┴──────────┴────────────┘

# Format các phân vùng:
mkfs.fat -F32 /dev/sda1    # EFI
mkswap /dev/sda2             # Swap
mkfs.ext4 /dev/sda3          # Root
3
Mount và cài base system
Pacstrap
# Mount phân vùng root:
mount /dev/sda3 /mnt

# Mount EFI:
mkdir -p /mnt/boot/efi
mount /dev/sda1 /mnt/boot/efi

# Kích hoạt swap:
swapon /dev/sda2

# Chọn mirror server nhanh nhất:
reflector --country Vietnam,Singapore --sort rate --save /etc/pacman.d/mirrorlist

# Cài hệ thống cơ bản:
pacstrap -K /mnt base linux linux-firmware
# Thêm packages tuỳ chọn:
pacstrap -K /mnt base linux linux-firmware \
  linux-headers base-devel \
  networkmanager vim nano sudo \
  grub efibootmgr \
  intel-ucode  # hoặc amd-ucode cho AMD
4
Cấu hình hệ thống
System Config
# Tạo fstab:
genfstab -U /mnt >> /mnt/etc/fstab
cat /mnt/etc/fstab  # Kiểm tra

# Chroot vào hệ thống mới:
arch-chroot /mnt

# Cài timezone:
ln -sf /usr/share/zoneinfo/Asia/Ho_Chi_Minh /etc/localtime
hwclock --systohc

# Cài locale (ngôn ngữ):
vim /etc/locale.gen
# Bỏ comment dòng: en_US.UTF-8 UTF-8
#                  vi_VN.UTF-8 UTF-8
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf

# Đặt hostname:
echo "archlinux" > /etc/hostname

# Cấu hình /etc/hosts:
cat >> /etc/hosts << EOF
127.0.0.1  localhost
::1        localhost
127.0.1.1  archlinux.localdomain archlinux
EOF
5
Tạo user và cài GRUB
User & Bootloader
# Đặt password root:
passwd

# Tạo user mới:
useradd -m -G wheel -s /bin/bash username
passwd username

# Cấp quyền sudo cho wheel group:
visudo
# Bỏ comment dòng: %wheel ALL=(ALL:ALL) ALL

# Cài bootloader GRUB (UEFI):
grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg

# Bật NetworkManager:
systemctl enable NetworkManager
6
Cài Desktop Environment
GUI Setup
# Thoát chroot và reboot:
exit
umount -R /mnt
reboot

# ─── Sau khi reboot và đăng nhập ───

# Cài KDE Plasma (khuyến nghị):
sudo pacman -S plasma kde-applications sddm
sudo systemctl enable sddm

# HOẶC cài GNOME:
sudo pacman -S gnome gnome-extra gdm
sudo systemctl enable gdm

# HOẶC cài XFCE (nhẹ):
sudo pacman -S xfce4 xfce4-goodies lightdm lightdm-gtk-greeter
sudo systemctl enable lightdm

# Khởi động lại để vào giao diện đồ họa:
sudo reboot
// Sau khi cài xong Cài AUR helper (yay): git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si — cho phép bạn truy cập kho AUR với hàng chục nghìn packages cộng đồng.
